Drop the following PNGs in this folder before shipping:

  icon-192.png            192x192
  icon-512.png            512x512
  icon-512-maskable.png   512x512 with safe zone for Android adaptive icons

Any monochrome wave/speaker silhouette on the SoundTide brand color (#1F3A5F)
works. The app will degrade gracefully if these are missing — only the home
screen install prompt looks slightly off without them.
